## 命令介绍

- `mkdocs new [项目名称]` - 创建一个项目，如果 `项目名称` 为 `.` 则默认为当前目录.
- `mkdocs serve` - 启动一个 live-reloading 的服务器，自动刷新页面.
- `mkdocs build` - 构建项目，生成 `site` 目录.
- `mkdocs -h` - 显示帮助信息.

## 项目结构

    mkdocs.yml    # 主要的配置文件.
    docs/ # 存放文档的目录
        index.md  # 这个文件是网站的主页.
        ...       # 其他的 markdown 文件，或者其他存放文档/图片/视频/音频的文件夹.
