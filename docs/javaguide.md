## 数据库

- 什么是数据库, 数据库管理系统, 数据库系统, 数据库管理员?

数据库是一个存储管理信息的仓库  
数据库管理系统是一个软件，用于建立，维护和管理数据库  
数据库系统是数据库，数据库管理系统和数据库管理员的集合  
数据库管理员是负责管理和操作数据库系统的人员

- 什么是元组, 码, 候选码, 主码, 外码, 主属性, 非主属性？

数据库表的一行叫做一个元组  
数据库表的一列叫做一个码  
能够唯一标识一个元组的码,而其子集不能,那么这组码就叫做候选码  
主码是从候选码中选出的某一个  
主属性是候选码中的码  
非主属性是不在任何一个候选码中的码

- 什么是 ER 图？

ER图也叫做实体关系图,由实体,属性和关系组成  
其中实体是指现实世界中的某个对象或者逻辑对象  
属性是指实体具有的某种属性  
关系是指实体与实体之间的联系,可以分为一对一关系,一对多关系和多对多关系

- 数据库范式了解吗?

第一范式,属性不可再分  
函数依赖:在属性X确定的情况下,必定能确定Y的值,那么就说Y函数依赖于X  
第二范式,消除非主属性对码的函数依赖  
传递函数依赖:假设X,Y,Z是三个不同的属性集合,X确定Y,Y确定Z,那么就说Z传递依赖X  
第三范式,消除非主属性对码的传递依赖  

- 主键和外键有什么区别?

主键是唯一标识一个元组的码  
外键是另一张表的主键,用来与其他表之间建立联系

- 为什么不推荐使用外键与级联？

真的不推荐吗?  
级联操作方便,减轻了程序代码量  
但是不适用于分布式和高并发场景  
因为对数据库的增删查改会导致级联更新,存在发生数据库更新风暴的风险

- 什么是存储过程?

存储过程是一些SQL语句的集合，一旦调试完成后就能稳定运行。

- drop、delete 与 truncate 区别？

三者用法不同：   
drop table 表名，将表和数据一起删除，truncate table 表名，只会清空表中数据，delete from table where 列名=值，删除表中符合条件的值  
属于不同的数据库语言：  
delete可以回滚（DML语句），而drop,truncate不行（DDL语句）  
三者速度不同：  
drop > truncate > delete,因为drop是直接释放内存，truncate清除表格后，还需要将索引恢复，delete最慢的原因是需要产生数据库日志用于回滚  

- 数据库设计通常分成几步？

1.需求分析：分析用户的需求，比如功能需求和性能需求  
2.概念结构设计：使用E-R图进行设计  
3.逻辑结构设计：将E-R图转换成表格，实习E-R模型到关系模型的转换。
4.物理结构设计：为设计的数据库选择合适的存储结构和存储路径  
5.编码实现  
6.数据库的运行和维护  

- SQL 和 NoSQL 有什么区别？

非关系型数据库，主要是键值对、文档以及图形类型数据存储

## MySQL 常见面试题

- 什么是关系型数据库？

关系数据库就是建立在关系模型上的数据库，关系模型表明了数据库中存储数据之间的联系（一对一，一对多，多对多）

- 什么是 SQL？

SQL是一种查询语言，目的是用于与数据库交互，从数据库中读写数据。  
其他功能：新建数据库、数据表、字段；在数据库中增加，删除，修改，查询数据；新建视图、函数、存储过程；对数据库中的数据进行简单的数据分析；搭配 Hive，Spark SQL 做大数据；搭配SQLFlow 做机器学习；

- 什么是 MySQL？

MySQL是关系数据库的一种，主要用于持久化存储系统中的一些数据。

- MySQL 有什么优点

1.成熟稳定，功能完善  
2.开源免费，生态完善  
3.兼容性强，支持常用的操作系统，支持多种开发语言  
4.事务支持优秀，InnoDB存储引擎默认使用repeatable-read,不会有性能损失与幻读问题。

- MySQL 字段类型

数值类型：整形，浮点型，定点型  
字符串类型：CHAR,VARCHAR,TEXT类，BLOB类  
时间类型：YEAR,DATE,TIME,DATETIME,TIMESTAMP

- 整数类型的 UNSIGNED 属性有什么用？

这个符号表示不允许负值的无符号整数，可以让正整数的上限提升一倍，因为他不需要存储负值

- CHAR 和 VARCHAR 的区别是什么？

CHAR是定长字符串，存储时会在右边填充空格，检索时会去掉空格。    
VARCHAR是变长字符串，需要消耗额外字节用于存储字符串长度。

- VARCHAR(100)和 VARCHAR(10)的区别是什么？

两者都是变成类型，分别表示能存储100个字符和10个字符。

- DECIMAL 和 FLOAT/DOUBLE 的区别是什么？

DECIMAL是定点数，后者是浮点数，定点数可以存储精确的小数值，而浮点数存储的是近似的小数值。

- 为什么不推荐使用 TEXT 和 BLOB？

TEXT用于存储长文本，BLOG则用于存储二进制大对象（图片，音视频）,两者检索效率低，可能会消耗大量网络和IO带宽，会导致表上的DML操作变慢。

- DATETIME 和 TIMESTAMP 的区别是什么？

DATETIME没有时区信息，而TIMESTAMP有时区信息  
DATETIME消耗8个字节存储空间，而TIMESTAMP消耗4个存储空间，故而后者表示的时间范围更小一些。

- NULL 和 '' 的区别是什么？

NULL表示缺少或未知的数据，而后者代表已知的空字符  
任何值与 NULL 进行比较（例如 =, !=, >, < 等）的结果都是 NULL  


- Boolean 类型如何表示？

使用TINYINT（1）表示。

- 手机号存储用 INT 还是 VARCHAR？

使用VARCHAR类型，一方面，手机号可能表示国家代码前缀，另一方面，VARCHAR可以根据手机前缀进行匹配，而int类型进行前缀匹配需要进行复杂函数转换。最后，手机号时常需要加密存储，加密存储后的结果是一串长字符，int无法存储。
